USD 비용 처리 흐름

1

회사 선택 (HOI)

companies.currency = "USD" 감지 → 폼이 USD 입력 모드로 전환

2

환율 조회

GET /api/exchange-rate?currency=USD → 한국수출입은행 API 호출 (4시간 캐시)

3

금액 입력

$ 입력 → 실시간으로 KRW 환산 표시 ("적용 환율: 1 USD = 1,385.50원")

4

제출 → DB 저장

amount = KRW 환산값 · amount_original = USD 센트 · exchange_rate = 환율 · currency = "USD"

5

표시

리스트/상세/관리자 페이지: $1,500.00 (₩2,078,250) 형식

6

Slack 알림

회사별 채널로 전송 · $1,500.00 (₩2,078,250) 병기

7

리포트 집계

amount(KRW) 기준으로 통합 집계 → 통화 무관 동일 단위 비교 가능

핵심 비즈니스 룰

  • 법카사용: 제출 즉시 APPROVED, 첨부 선택, 승인 불필요
  • 입금요청: 제출 시 SUBMITTED, 첨부 필수, ADMIN 승인 필요
  • 회사 권한: 코리아 소속 → 다른 회사(리테일/HOI) 비용 제출 가능 / 그 외 → 자기 회사만
  • 통화 결정: 회사의 currency 컬럼이 결정 (HOI=USD, 나머지=KRW)
  • 환율 저장: USD 비용 제출 시점의 환율을 exchange_rate에 고정 저장
  • 리포트: 모든 통화는 amount(KRW) 컬럼 기준으로 집계
  • RBAC: MEMBER / ADMIN 2단계, Supabase RLS + middleware
