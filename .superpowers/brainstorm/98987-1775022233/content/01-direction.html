리포트 기능 방향

현재 있는 기능과 비교하여, 어떤 방향으로 진행할까요?

현재 구현된 기능

  • Admin 대시보드 — 월별 추이, 카테고리 파이차트, Top 5 제출자
  • Admin 리포트 — 날짜/유형/상태/카테고리 필터 + 부서별 비용 + CSV 다운로드
A

기존 리포트 페이지 대폭 개선

더 많은 차트, 고급 필터, 기간 비교, 트렌드 분석 등을 기존 /admin/reports 페이지에 추가

B

새로운 분석 대시보드 페이지 추가

기존 리포트와 별도로 새 페이지를 만들어 심층 분석 기능 제공

C

둘 다 — 기존 개선 + 새 기능 추가

기존 리포트 페이지를 개선하면서, 추가 분석 페이지도 새로 구축
