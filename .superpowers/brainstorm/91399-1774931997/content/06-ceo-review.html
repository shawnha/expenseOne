핵심 피드백

1. 법인카드 테이블 누락
Design Decisions에 "법인카드 분리"를 명시했으나 실제 모델에 빠짐
2. 리포트/대시보드 회사별 집계 누락
"코리아 이번 달 얼마 썼어?" — CEO가 가장 많이 볼 화면인데 스펙에 없음
3. 기존 사용자 마이그레이션 UX 부재
기존 20~30명이 로그인했을 때 소속 회사 선택 플로우 필요
4. 다른 회사 경고 → 부드러운 라벨로
주황색 경고 대신 회사명 배지 + "(소속 외)" 텍스트만
5. 회사 관리 스코프 축소
CRUD 페이지 → /admin/settings 내 편집 섹션으로 통합
6. 벌크 재분류 기능
기존 비용 중 리테일 건을 일괄 회사 변경할 수 있는 도구
7. 롤백 계획 명시
마이그레이션 스크립트에 롤백 SQL 포함
